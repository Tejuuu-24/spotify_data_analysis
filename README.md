# spotify_data_analysis
data analytics project showcasing spotify data analysis using python and power Bi

1. Exploratory Data Analysis (Python)
Performed analysis to understand feature distribution and relationships
Univariate analysis: histograms, boxplots
Bivariate analysis: scatter plots, correlation heatmaps
Generated summary statistics (mean, median, variance, skewness, kurtosis)

2. Data Cleaning & Preprocessing
Removed duplicate rows
Handled missing values using mean/median/mode
Treated outliers using IQR and Winsorization
Encoded categorical features using dummy variables
Applied standardization and normalization to numerical columns
Created the final cleaned dataset for Power BI

3. Power BI Dashboard
KPIs Included:
Most Followed Artist
Total Tracks
Total Artists
Average Track Popularity
Visuals Added:
Top Tracks by Popularity (Bar Chart)
Top Artists by Followers (Donut Chart)
Slicers for Interactive Filtering:
Artist
Genre
Power BI allows users to filter by artist or genre and updates all visuals automatically.

4. Insights Extracted
Identified top-performing artists based on follower count
Observed popularity distribution across tracks
Compared trends across different genres
Analyzed artist performance under various filters

5. Tech Stack
Python: Pandas, NumPy, Matplotlib, Seaborn
Power BI: Dashboard building, DAX measures
Data: CSV/Excel (Kaggle Spotify dataset)
